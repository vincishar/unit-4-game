Crystal Collector Game!

    You will be given a random number at the start of the game
    There are four crystals below. By clicking on a crystal you will add a specific amount of points to your total score
    You win the game by matching the target score to the number from the crystals below. You lose the game if you go above the target score.
    The number of each crystal is hidden from you until you click on it.
    Every time when the game starts, the game will change the number of each crystal.
